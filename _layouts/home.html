Latest Project

HedgeShot is a First Person Shooter game created with Unreal Engine 5. It is a month-long game jam effort for the GDFG TEAM JAM #4, with a team of four including me. It takes place in a hedge-maze like environment where you fight your way through zombie hordes. In this endeavour, I assume the role of game designer as well as level designer and the director.
